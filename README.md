# Academic Portal (using CodeIgniter 4)

An academic portal built with **CodeIgniter 4**, **MySQL**, and **Bootstrap 5** that supports **role-based authentication** for **Admin** and **Students**.  
Students can register, log in, view, and enroll in courses, while Admins can manage courses and student records.


## 🚀 Features

### Authentication
- Student registration with **NIM as password (hashed)**.
- Login via **NIM** or **Email**.
- Admin login (default: `admin` / `password246`).
- Session-based authentication with role management.

### Student Role
- Browse available courses.
- Enroll in courses.
- View "My Courses" dashboard.
- Profile includes name, NIM, and enrolled courses.

### Admin Role
- Manage courses (Create, Read, Update, Delete).
- Manage student data.
- View enrolled courses per student.
